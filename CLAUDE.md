# Project: CoM2 Damage Calculator

## Key Files
- `MoM Combat Mechanics Reference.md` — Condensed combat mechanics reference. This is the primary source for how combat works.
- `CoM1 or MoM material/Fandom site/` — Raw wiki source files. Only read these if the mechanics reference needs to be updated or clarified.

## Rules
- Do NOT read files under `CoM1 or MoM material/` unless actively editing `MoM Combat Mechanics Reference.md`.
- Use exact probability distributions (binomial math), not Monte Carlo simulation.
- MoM uses 10% increments for To Hit / To Block. CoM2 uses 1% increments (Phase 4).

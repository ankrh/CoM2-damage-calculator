# Enchantment Unit Type Restrictions by Version

This table documents which unit types (Normal, Fantastic, Hero) can be affected by each enchantment in MoM 1.31, CoM 6.08, and CoM2 1.05.

**Legend:**
- ✓ = Can be cast on this unit type
- ✗ = Cannot be cast on this unit type
- • = Combat enchantment that affects all eligible units on battlefield regardless of type
- ~ = Global enchantment (affects all wizard's creatures or all units on battlefield)
- Table columns: N = Normal units | F = Fantastic units | H = Hero units
- Format example: ✓ ✓ ✓ = can cast on Normal, Fantastic, and Hero; ✗ ✓ ✗ = Normal units only

## Key Findings

### Restrictions by Version

**MoM 1.31/1.60:**
- "Normal units only" restrictions: Holy Weapon, Holy Armor, Flame Blade, Chaos Channels, Eldritch Weapon, Black Channels
- These were likely intended to limit transformation/mutation effects to generic troops, not unique units
- Combat enchantments like Prayer, Black Prayer, Metal Fires, and Blur affect all units on the battlefield
- Shatter targeted all unit types in MoM (restriction was added in CoM/CoM2)

**CoM 6.08:**
- No reliable source for CoM 6.08 spell targeting. CoM2 values are assumed to apply here too.
- Metal Fires and Eldritch Weapon were removed (not present in CoM2 helptext)
- New spells added: Blazing March (combat enchantment), Blood Lust, Mystic Surge, Focus Magic, Survival Instinct (global enchantment)

**CoM2 1.05:**
- "Normal unit only" restrictions retained on Holy Weapon, Holy Armor, Flame Blade, and Chaos Channels (confirmed from CoM2 helptext)
- Adds CoM2-only spells: Discipline, Ruler of Underworld, Destiny, Inner Power, Mislead, Reinforce Magic
- Some MoM-only spells (Metal Fires, Eldritch Weapon, Black Channels) remain removed

### Combat vs Unit Enchantments

**Combat Enchantments** (affect entire battlefield):
- Prayer, High Prayer, Black Prayer, Metal Fires, Blur, Blazing March
- These are cast during combat and affect all appropriate units (friendly/enemy)
- Do NOT require targeting individual units

**Unit Enchantments** (cast on individual units):
- All others listed above
- Must be cast on a friendly unit unless explicitly marked as targeting enemies
- Subject to unit type restrictions (where noted)

### Restrictions Summary

| Type | Count | Examples |
|---|---|---|
| Any unit type (N\|F\|H) | 20+ | Bless, Lionheart, Haste, Magic Immunity, Immolation, Weakness, Guardian Wind, etc. |
| Normal units only (N only) | 6 | Holy Weapon, Holy Armor, Flame Blade, Chaos Channels, Shatter (CoM+), Heroism (CoM+) |
| Normal non-hero only | 2 | Discipline (CoM2), Destiny (CoM2) |
| Combat enchantment (•) | 6 | Prayer, High Prayer, Black Prayer, Metal Fires, Blur, Blazing March |
| Global enchantment (~) | 2 | Charm of Life, Survival Instinct |
| Version-specific | 6+ | Blood Lust (CoM+), Discipline (CoM2), Inner Power (CoM2), etc. |

## Important Notes

1. **MoM "Normal Unit Only" Spells:** The six spells with "normal unit only" restrictions in MoM (Holy Weapon, Holy Armor, Flame Blade, Chaos Channels, Eldritch Weapon, Black Channels) were likely designed this way to:
   - Prevent transformation of heroes and fantastic units
   - Keep the spell balanced (transforming a powerful fantastic creature would be overpowered)
   - Maintain game balance by requiring normal units for mutation effects

2. **CoM/CoM2 "Normal Unit Only" Spells:** Holy Weapon, Holy Armor, Flame Blade, and Chaos Channels remain normal-unit-only in CoM2 (confirmed from helptext). The restriction language is "original unit type is a normal unit", meaning units transformed to fantastic by another spell still qualify.

3. **Combat vs. Unit Enchantments:**
   - **Combat Enchantments** (Prayer, Black Prayer, Blur, Metal Fires, Blazing March): Cast during battle and affect the entire battlefield
   - **Global Enchantments** (Charm of Life, Survival Instinct): Persistent overland effects on all wizard's creatures
   - **Unit Enchantments** (all others): Must be individually cast on friendly units; some are curses affecting enemy units

4. **Data Limitations:**
   - CoM 6.08 has no reliable source for unit type targeting; its column is assumed to match CoM2
   - CoM2 values are derived from `Manuals/CoM2 spells helptext.txt` which has explicit "Target:" fields

5. **Heroes and "Fantastic" Units:**
   - "Fantastic" units = Summoned creatures (not player-built normal troops)
   - Heroes = Special single-unit commanders
   - In MoM, "fantastic creatures and heroes may not be possessed" (per Possession spell documentation)
   - MoM's "normal unit only" spells appear intentionally designed to exclude summoned creatures
